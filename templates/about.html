Tentang Project

Klasifikasi Jenis Pakaian Menggunakan Transfer Learning Xception

Deskripsi Project

ClothVision adalah aplikasi web berbasis Flask yang mampu mengklasifikasikan jenis pakaian dari sebuah gambar menggunakan teknik Transfer Learning dengan arsitektur Xception. Project ini dikembangkan sebagai bagian dari Tugas 12 Praktikum Sistem Multimedia, yang menggabungkan konsep Computer Vision, Deep Learning, dan pengembangan aplikasi web.

Model dilatih menggunakan bobot ImageNet sebagai basis (base model dibekukan/frozen), kemudian ditambahkan lapisan classifier khusus untuk mengenali 10 kelas pakaian.

Teknologi yang Digunakan

Python 3.11
TensorFlow 2.x
Keras
Flask
Bootstrap 5
Xception (Transfer Learning)
Pillow
NumPy & Pandas
Matplotlib
Scikit-learn

Dataset

Dataset yang digunakan adalah Clothing Dataset Small yang terdiri dari 10 kelas pakaian:

Dress Hat Longsleeve Outwear Pants Shirt Shoes Shorts Skirt T-Shirt

Sumber dataset: Kaggle - Clothing Dataset Small oleh Abdelrahman Soltan.

Arsitektur Model

  1. Base Model: Xception (include_top=False, weights='imagenet')
  2. Base model dibekukan (frozen) sepenuhnya
  3. GlobalAveragePooling2D
  4. Dropout(0.5)
  5. Dense(256, activation='relu')
  6. BatchNormalization()
  7. Dropout(0.3)
  8. Dense(10, activation='softmax')
